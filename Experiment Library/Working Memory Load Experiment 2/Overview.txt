Experiment:	Updating_VWM_SRP_October_2020

This experiment, is assessing the effect of loading information in
Visual Working Memory (VWM) on distractability of task-irrelevant visual information.

Please note that this is a Reaction Times (RTs) experiment, so please try to be as quick 
and as accurate as possible in making your responses.

Please fixate the central fixation.
 
If you need to blink, you can do it after the feedback.
